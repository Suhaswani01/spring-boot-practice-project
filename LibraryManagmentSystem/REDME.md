# 📚 Library Management System

## Built With
- Java
- Spring Boot MVC
- Thymeleaf
- Maven

## Features
- View All Books
- Add New Book
- Search Book by ID

## How to Run
1. Clone the project
2. Open in IDE
3. Run LibraryAppApplication.java
4. Open browser: localhost:8080/library/home

## Project Structure
- controller/ → BookController.java
- model/      → Book.java
- templates/  → HTML pages
