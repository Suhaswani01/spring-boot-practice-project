# 🌱 Spring Boot Mini Projects — Sessions 5, 6 & 7

Practice projects built while learning Spring Boot Core concepts.

---

## 📁 Projects

### ✅ MiniProject01 — Spring Boot Basics
- `@SpringBootApplication`, `@Component`, `@Service`, `@Autowired`
- CommandLineRunner se console output
- IOC Container aur Bean lifecycle



### ✅ MiniProject03 — JDBC + MySQL (Student Management)
- MySQL database connection with HikariCP connection pooling
- `JdbcTemplate` se INSERT aur SELECT operations
- Layered architecture: Model → Service Interface → ServiceImpl → Runner
